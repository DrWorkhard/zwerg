Isometric Browser Game
======================

Project Overview
----------------
This project aims to create an isometric game that runs in the browser. The visual style will be inspired by *The Legend of Zelda: A Link to the Past*.

The goal is to demonstrate a minimal game loop with JavaScript handling the logic. Rendering is done using the built in `<canvas>` element.

Running the Example
-------------------
Open `index.html` in any modern web browser. A small map will appear, and you can move the red diamond with **WASD**.

Future Plans
------------
Core components like AI and advanced rendering can be expanded later. For now the repository serves as a base for further development.
